# Earcillator

Implementation accompanying the paper:
**“From Transmission Line Models to Stuart–Landau Oscillators:
A Hopf-Based Description of Cochlear Dynamics”**

---

## 🌐 Project Page

https://SPSC-TUGraz.github.io/earcillator/

---

## 💻 Repository Structure

```text
docs/        Project webpage (GitHub Pages)
src/         Core model implementation
```

---

## 🚀 Usage

Clone the repository:

```bash
git clone https://github.com/SPSC-TUGraz/earcillator.git
cd earcillator
```

Further usage depends on the scripts and modules in `src/`.
